Constructs a quadratic function $ q(t)=a+bt+ct^2$, with
\begin{DoxyItemize}
\item $ a = L(x)+\nu L'(x)dn+\frac{\nu^2}{2}L''(x)(dn)^2 $
\item $ b = L'(x)dt + \nu L''(x)(dn,dt) $
\item $ c = \frac{1}{2}L''(x)(dt)^2 $
\end{DoxyItemize}
