Constructs the quadratic model of a Lagrange functional for a composite step method.

Parameters
nunormal step damping factor
dnnormal step
dttangential step
LLagrange functional
xcurrent iterate

Constructs a quadratic function \( q(t)=a+bt+ct^2\), with

  • \( a = L(x)+\nu L'(x)dn+\frac{\nu^2}{2}L''(x)(dn)^2 \)
  • \( b = L'(x)dt + \nu L''(x)(dn,dt) \)
  • \( c = \frac{1}{2}L''(x)(dt)^2 \)
Returns
\( q(t)=a+bt+ct^2\)
