# FSA - Function Space Algorithms

### Download
Download via
  - git clone https://github.com/lubkoll/FSA.git

### Installation
Go to download folder, i.e.
  - cd FSA
  
and type

  - mkdir build
  - cd build
  - cmake ..
  - make
