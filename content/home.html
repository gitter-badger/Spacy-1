Welcome to Spacy,

an abstraction layer for sharing and developing vector space algorithms. It is motivated by the following ideas:

Sharing

The main aim of this library is to facilitate sharing of algorithms. With Spacy researchers and users can develop and share algorithms without being tied to specific frameworks for the problem definition.

Cleaner Code

The separation of problem definition and solution algorithm helps in revealing the algorithmic structure in its implementation. Moreover it helps enforcing the separation of responsibilities.

Rapid Prototyping

Type erasure is used for the definition of interfaces. Consequently duck-typing and value-semantics are supported.
Developing algorithms in C++ was never easier.
