Dependencies

Spacy depends on Boost.TypeErasure (available from Boost Version 1.54.0).

In Ubuntu you can install boost with

sudo apt-get install libboost-dev

Installation

Spacy can be easily installed with cmake. Boost.TypeErasure and, if present, FEniCS (dolfin) should be detected automatically.

The usual installation procedure is:

  • cd spacy/root/dir
  • mkdir build
  • cd build
  • cmake ..
  • make
