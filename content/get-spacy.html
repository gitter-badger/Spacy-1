Dependencies

Spacy itself does not require anything than a recent C++-compiler. For testing googletest is used and will be downloaded by cmake.

Installation

Spacy can be easily installed with cmake. The usual installation procedure is:

  • cd spacy/root/dir
  • mkdir build
  • cd build
  • cmake ..
  • make

Libraries that are build with cmake, such as Eigen or FEniCS, should be detected by cmake. For usage with Kaskade7 run cmake -DKaskade=ON ... For building the tests you have to specify cmake -DBuildTest=ON ...
